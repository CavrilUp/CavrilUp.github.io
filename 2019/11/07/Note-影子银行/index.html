Note | 影子银行

Author: Cavril Date: 十一月 7, 2019  22:24:15 Category: Note

资金池

滚动募集、混合运作、分离定价、纳入银行的存贷业务都要记入资产负债表

总结

影子银行是不在银行资产负债表的存贷款项中体现，却做着银行存贷业务的“影子部队”

  1. 滚动募集：资金非一次性募集完毕，二是通过分期发现或开放申购的方式在不同时点分别完成募集
    期限错配是通过滚动募集实现的
    滚动募集的两种模式：
    a. 每笔资金的投资与项目回款一一对应，后续滚动募集的资金并不用于兑付已经到期的续金
    b. 前一期已募集资金到期时，就把后一期资金兑付前一期的资金，使已到期的资金实现退出（流动性风险）
  2. 混合运作（集合运作）：受托人将不同时期发现产品募集的资金进行集中，并统一核算，统一管理使用，资金来源和资产运用没有明确一一对应（存在资金挪用，违法违规 ，携款跑路的风险）
  3. 分离定价（根本属性）：内在价格≠认购价格
    导致产品的资金申购或赎回时未按照规定进行合理估值，脱离对应标的资产的实际收益率确定投资者申购价格及赎回，到期时的收益水平。

一些形式：（for bank）

1.表内-流动性、资本限制，银行不能贷款给借款人
投向行业（房地产、地方融资平台、两高一剩）借一个非银金融机构（如信托-不受额度、投向限制），银行就进行买入返售该机构的金融资产或应收款项投资
2.表外-银行将理财资金通过投资非标等资产投放给借款企业，但现在最终投资的底层资产实现穿透式管理，因此此路径已经行不通了

资管产品属于代客理财，发行人对资管产品进行刚性兑付-银行存款
影子银行：记录在同业资产，应收款项
